# Contracting Project Management System (CPMS)

A specialized system designed to manage contracting projects, track work progress, and organize project resources and financials.

## Key Features
- Project and task management
- Managing key elements and items
- Project contract management
- Work progress tracking and reporting
- Resource and workforce management
- Cost estimation and expense tracking
- Project-based financial reports
- Multi-level approval workflows
- Project Works Inventory Management
- Project payment certificate management

## Technologies
- ASP.NET Web Forms
- C#
- SQL Server
- RDLC Reports
- JavaScript, Bootstrap 5
- HTML, CSS

## My Contribution
- Designed project workflow and task tracking modules
- Developed reporting system for project performance
- Implemented cost tracking and financial summaries
- Structured database for project-based operations

## Availability
Private repository — Developed for a private company — Source code is not publicly available.

# نظام إدارة مشاريع المقاولات (CPMS)

نظام متخصص لإدارة مشاريع المقاولات ومتابعة تقدم الأعمال وتنظيم الموارد والتكاليف.

## المميزات الرئيسية
- إدارة المشاريع والمهام
- ادارة البنود الرئيسية والبنود الفرعية
- ادارة عقود المشاريع
- متابعة تقدم العمل وإعداد التقارير
- إدارة الموارد والعمالة
- تقدير التكاليف ومتابعة المصروفات
- تقارير مالية لكل مشروع
- نظام موافقات متعدد المستويات
- ادارة حصر اعمال المشاريع
- ادارة مستخلصات المشاريع

## التقنيات المستخدمة
- ASP.NET Web Forms
- C#
- SQL Server
- RDLC Reports
- JavaScript, Bootstrap 5
- HTML, CSS

## دوري في المشروع
- تصميم نظام إدارة المشاريع والمهام
